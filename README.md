# traffic-light
This website was made for Hack Club's Boba Teed Up program! I created a simple traffic light animation. It turns on the green light, yellow light, and then red light in a realistic timing and then repeats forever. The challenge was to use only HTML and CSS without JS or frameworks. This website took me approximately 1 hour to make. I used W3Schools to learn how to use @keyframes to change color in animations.
